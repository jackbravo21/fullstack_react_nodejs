Este projeto é FullStack, FrontEnd e BackEnd, com sistema completo de login, visualização de usuários, produtos, um CRUD avançado com dados, imagens e cokies/tokens;


No React, são utilizados:
- Padrões de projetos com regras de negócios separadas;
- Rotas e ProtectedRoutes: react-router-dom;
- Estados: useState, useEffect;
- Redux: Reducers, Saga, Store, Slice;
- Tokens e Cookies;
- Envio de arquivos;
- Axios;

No NodeJs, são utilizados:
- Configuração profissional do servidor;
- Padrões de projetos com regras de negócios separação das obrigações;
- Rotas protegidas com Middlewares;
- Criptografia;
- Multer (recebimento de arquivos);
- Sequelize com Mysql;
- Tokens e Cookies;
